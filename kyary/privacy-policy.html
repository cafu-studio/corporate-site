KYARYプライバシーポリシー

最終更新日: 2025年7月

はじめに

合同会社Cafu（以下「当社」といいます。）は、お客様の個人情報保護の重要性を認識し、アプリケーション「KYARY」（以下「本アプリ」といいます。）におけるお客様の個人情報の取扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます。）を定めます。

第1条（個人情報の定義）

「個人情報」とは、個人情報保護法にいう「個人情報」を指すものとします。

第2条（収集する情報）

当社は、本アプリの提供にあたって、以下の情報を収集します。

2.1 ユーザーが提供する情報

  1. アカウント情報
    • Login with AmazonまたはLogin with Appleを通じて取得する認証情報
    • ユーザーが設定した表示名（ニックネーム）
  2. コンテンツ情報
    • Kindleから取得したハイライト情報
    • ユーザーが作成した質問と回答の履歴
    • お気に入り登録したハイライト
    • 書籍情報（タイトル、著者名、ASIN）

2.2 自動的に収集される情報

  1. 利用状況データ
    • アプリの利用頻度、利用時間
    • 機能の使用状況
    • エラーログ
  2. デバイス情報
    • デバイスの種類、OSバージョン
    • アプリのバージョン
    • デバイスID（広告識別子は収集しません）

第3条（情報の利用目的）

当社は、収集した情報を以下の目的で利用します。

  1. サービスの提供・運営
    • ユーザー認証およびアカウント管理
    • ハイライトの保存、管理、検索機能の提供
    • AI質問応答機能の提供
    • サブスクリプションの管理
  2. サービスの改善・開発
    • 利用状況の分析によるサービスの改善
    • 新機能の開発
    • ユーザーエクスペリエンスの向上
  3. カスタマーサポート
    • お問い合わせへの対応
    • 技術的な問題の解決
  4. その他
    • 利用規約違反の調査、対応
    • 法令に基づく対応
重要：当社は、Login with AmazonおよびLogin with Appleを通じて取得した認証情報を、ユーザーのログイン認証のためにのみ使用し、その他の目的では使用しません。

第4条（第三者サービスとの連携）

本アプリは、以下の第三者サービスと連携しています。

4.1 認証サービス

  1. Amazon Cognito
    • ユーザー認証の管理
    • ユーザーIDの生成と管理
    • セッション管理
  2. Login with Amazon / Login with Apple
    • ソーシャルログイン機能の提供
    • 認証情報の取得

4.2 その他のサービス

  1. RevenueCat
    • サブスクリプションの管理
    • 購入履歴の管理
    • 収益分析（個人を特定しない形式）
  2. Amplitude
    • 利用状況の分析
    • ユーザー行動の分析（個人を特定しない形式）

これらの第三者サービスは、それぞれ独自のプライバシーポリシーに基づいて情報を取り扱います。

4.3 第三者への提供データ

  1. RevenueCatへ提供するデータ
    • ユーザーID（匿名化されたID）
    • 購入履歴
    • サブスクリプション状態
  2. Amplitudeへ提供するデータ
    • ユーザーID（匿名化されたID）
    • アプリ内の行動データ
    • デバイス情報

第5条（データ処理の法的根拠）

当社は、以下の法的根拠に基づいて個人情報を処理します。

  1. 契約の履行：サービス提供のために必要な処理
  2. 同意：マーケティング目的での処理
  3. 正当な利益：サービス改善、不正利用防止のための処理
  4. 法的義務：法令に基づく処理

第6条（個人情報の第三者提供）

当社は、次に掲げる場合を除いて、あらかじめユーザーの同意を得ることなく、第三者に個人情報を提供することはありません。

  1. 法令に基づく場合
  2. 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難であるとき
  3. 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難であるとき
  4. 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに対して協力する必要がある場合であって、本人の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがあるとき
  5. 予め次の事項を告知あるいは公表し、かつ当社が個人情報保護委員会に届出をしたとき
    • 利用目的に第三者への提供を含むこと
    • 第三者に提供されるデータの項目
    • 第三者への提供の手段または方法
    • 本人の求めに応じて個人情報の第三者への提供を停止すること
    • 本人の求めを受け付ける方法

第7条（個人情報の開示）

  1. 当社は、本人から個人情報の開示を求められたときは、本人に対し、遅滞なくこれを開示します。ただし、開示することにより次のいずれかに該当する場合は、その全部または一部を開示しないこともあり、開示しない決定をした場合には、その旨を遅滞なく通知します。
    • 本人または第三者の生命、身体、財産その他の権利利益を害するおそれがある場合
    • 当社の業務の適正な実施に著しい支障を及ぼすおそれがある場合
    • その他法令に違反することとなる場合
  2. 前項の定めにかかわらず、履歴情報および特性情報などの個人情報以外の情報については、原則として開示いたしません。

第8条（個人情報の訂正および削除）

  1. ユーザーは、当社の保有する自己の個人情報が誤った情報である場合には、当社が定める手続きにより、当社に対して個人情報の訂正、追加または削除（以下「訂正等」といいます。）を請求することができます。
  2. 当社は、ユーザーから前項の請求を受けてその請求に応じる必要があると判断した場合には、遅滞なく、当該個人情報の訂正等を行うものとします。
  3. 当社は、前項の規定に基づき訂正等を行った場合、または訂正等を行わない旨の決定をしたときは遅滞なく、これをユーザーに通知します。

第9条（個人情報の利用停止等）

  1. 当社は、本人から、個人情報が、利用目的の範囲を超えて取り扱われているという理由、または不正の手段により取得されたものであるという理由により、その利用の停止または消去（以下「利用停止等」といいます。）を求められた場合には、遅滞なく必要な調査を行います。
  2. 前項の調査結果に基づき、その請求に応じる必要があると判断した場合には、遅滞なく、当該個人情報の利用停止等を行います。
  3. 当社は、前項の規定に基づき利用停止等を行った場合、または利用停止等を行わない旨の決定をしたときは、遅滞なく、これをユーザーに通知します。
  4. 前2項にかかわらず、利用停止等に多額の費用を有する場合その他利用停止等を行うことが困難な場合であって、ユーザーの権利利益を保護するために必要なこれに代わるべき措置をとれる場合は、この代替策を講じるものとします。

第10条（データの保管とセキュリティ）

  1. 当社は、収集した個人情報を適切に管理し、不正アクセス、紛失、破損、改ざんおよび漏洩などを防ぐため、必要かつ適切なセキュリティ対策を実施します。
  2. 個人情報は、AWS（Amazon Web Services）のセキュアなクラウド環境に保管されます。
  3. 通信は全てSSL/TLSにより暗号化されます。
  4. アクセス権限は必要最小限の範囲で管理されます。

第11条（データ保存期間）

当社は、以下の期間、個人情報を保存します。

  1. アカウント情報：アカウント削除申請から30日後に完全削除
  2. コンテンツ情報：アカウント削除と同時に削除
  3. 利用状況データ：収集から2年間保存後、自動削除
  4. エラーログ：収集から90日間保存後、自動削除
  5. バックアップデータ：アカウント削除後90日以内に削除

第12条（アカウント削除）

  1. ユーザーは、お問い合わせ窓口（nakano@cafu.jp）へのメール連絡により、いつでもアカウントの削除を申請できます。
  2. 削除申請から7日以内にアカウントと関連データを削除します。
  3. アカウント削除後も、法的義務により保存が必要なデータは、必要な期間保存されます。

第13条（自動意思決定とプロファイリング）

  1. 当社は、AI技術を使用して以下の処理を行います。
    • ハイライトに基づく質問応答の生成
    • 関連するハイライトの検索と提案
    • 書籍の要約生成
  2. これらの処理は、ユーザーの利便性向上を目的として行われ、ユーザーに不利益を与える自動的な決定は行いません。

第14条（子どもの個人情報）

  1. 本アプリは13歳未満の子どもを対象としておらず、13歳未満の子どもから意図的に個人情報を収集することはありません。
  2. 13歳以上18歳未満のユーザーは、保護者の同意を得て利用するものとします。
  3. 保護者は、未成年者のアカウントに関して、データの開示、訂正、削除を要求する権利を有します。

第15条（Cookie およびトラッキング技術）

本アプリは、ウェブブラウザベースではないため、Cookieを使用しません。ただし、アプリ内で必要な情報を保存するために、デバイスのローカルストレージを使用することがあります。

第16条（国際的なデータ転送）

ユーザーの情報は、日本国内のサーバーで処理・保管されますが、サービスの提供に必要な場合、日本国外のサーバーに転送される可能性があります。その場合も、本ポリシーに準じた適切な保護措置を講じます。

第17条（匿名化・仮名化の処理）

  1. 当社は、統計データ作成やサービス改善のために、個人情報を匿名化または仮名化して使用することがあります。
  2. 匿名化されたデータは、個人を特定できない形で処理され、マーケティングや研究目的で使用されることがあります。
  3. 仮名化されたデータは、追加情報なしには個人を特定できない状態で管理されます。

第18条（緊急時の対応）

  1. 個人情報の漏洩、滅失、毀損等の事態が発生した場合、当社は速やかに以下の対応を行います。
    • アプリ内での告知
    • 当社ウェブサイトでの情報公開
  2. 緊急連絡先：nakano@cafu.jp

第19条（プライバシーポリシーの変更）

  1. 本ポリシーの内容は、法令の変更、サービスの変更、その他の事由により変更することがあります。
  2. 重要な変更（収集する情報の種類、利用目的、第三者提供の範囲等の変更）については、アプリ内通知により、変更の30日前までにお知らせします。
  3. 軽微な変更については、本アプリ上への掲載をもって通知に代えることができます。
  4. 変更後のプライバシーポリシーは、本アプリ上に掲載した日から効力を生じます。

第20条（お問い合わせ窓口）

本ポリシーに関するお問い合わせは、以下の窓口までお願いいたします。

  • 会社名：合同会社Cafu
  • 担当部署：個人情報管理責任者
  • メールアドレス：nakano@cafu.jp

第21条（法令遵守）

当社は、個人情報の取扱いに関して、個人情報保護法その他の法令を遵守します。
